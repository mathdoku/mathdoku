Source code

MathDoku is hosted on code.google.com.

You can grab the source code, and start hacking right away:

svn checkout http://mathdoku.googlecode.com/svn/trunk/ mathdoku
Localisation

If you can converse in a language that MathDoku doesn't speak yet, you can provide a localisation which will be included in the game. See translations for more information.

Privacy first

Android apps must explicitly request permission during installation to be able to access the internet. MathDoku doesn't make such a request, and therefore cannot send any information without your consent.

Google provides with very general statistics about de players to the developers (such as the percentage of Spanish speaking players). More specific information is not sent automatically.

Usage data

MathDoku might ask you to send data about the way you play the game. The developers are interested in how frequent the different functions are used. This information will be used to decide upon features to build in upcoming versions.

We encourage you to share this data with us, but you may choose not to do so if you wish. You can continue to play the game as normal either way.

Usage data is sent by email.

Privacy eerst

Android apps moeten expliciet toestemming vragen tijdens de installatie om het internet te kunnen gebruiken. MathDoku vraagt dit niet, en kan dus geen informatie opsturen zonder jouw instemming.

Google geeft de ontwikkelaars hele globale statistieken over de spelers (zoals het percentage Spaans sprekende spelers). Meer specifieke informatie wordt niet automatisch verstuurd.

Gebruiksdata

MathDoku kan vragen om data over jouw speelstijl op te sturen. De ontwikkelaars zijn geïnteresseerd in hoe vaak de functionaliteiten in het spel worden gebruikt. Deze informatie zal worden gebruikt bij het ontwerpen van functionaliteiten in toekomstige versies.

We moedigen je aan deze data met ons te delen, maar je kunt er voor kiezen dit niet doen. Je kunt hoe dan ook het spel gewoon blijven spelen.

Gebruiksdata wordt verstuurd per email.
